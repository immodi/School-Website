Help & Support

System Requirements

Using Smart School Zoom Live Class, you must have running Smart School version 6.3.0 or its newer version on your server. Below are a list of items you should ensure before using Smart School Zoom Live Class.

  • Smart School version 6.3.0 or its newer version pre installed on your server
  • Zoom account api key

Smart School Zoom Live Class Installation

Always backup your all files and database before installing Smart School Zoom Live Class

Smart School Zoom Live Class can be install in following 2 steps -

  1. Download Smart School Zoom Live Class from CodeCanyon. Now extract and open smart_school_zoom_live_class_src/step-1 folder.
  2. Import zoom_live_class_4.0_db_import.sql file in your Smart School database using phpmyadmin, while importing you must disable/uncheck Enable foreign key checks option at import page. After successfully import your Zoom Live Class database has been installed.
  3. After database installation, copy all the files under smart_school_zoom_live_class_src/step-2 folder and overwrite to under your Smart School installation folder.
  4. Clear the cache of your browser and login to Admin Panel and check if everything is okay.

If you experience any trouble in installation, open support ticket and we will be happy to assist you.

Smart School Zoom Live Class Update

Always backup your all files and database before updating Smart School Zoom Live Class

Smart School Zoom Live Class version 3.0 can be update to this latest version 4.0 in following 1 step -

  1. Download Smart School Zoom Live Class from CodeCanyon. Now extract and open smart_school_zoom_live_class_update_3.0_to_4.0/step-1 folder and overwrite to under your Smart School installation folder.
  2. Clear the cache of your browser and login to Admin Panel and check if everything is okay.

If you experience any trouble in installation, open support ticket and we will be happy to assist you.
